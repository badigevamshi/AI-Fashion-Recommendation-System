
# AI Fashion Recommendation System 👗🤖

An AI-powered fashion recommendation web application that analyzes user photos and provides personalized outfit suggestions using Computer Vision and Large Language Models.

## 🚀 Overview

The **AI Fashion Recommendation System** allows users to upload a photo and receive intelligent fashion suggestions based on their skin tone, body type, gender, and season.

The application uses **OpenCV for face detection and skin tone analysis** and **Groq LLM (Llama model)** to generate personalized fashion advice including clothing, footwear, accessories, and hairstyle recommendations.

---

## ✨ Features

* 📸 Upload a photo for fashion analysis
* 🧠 AI-powered outfit recommendations
* 🎨 Skin tone detection using OpenCV
* 👕 Suggestions for shirts, bottoms, footwear, accessories, and hairstyles
* 🌦 Seasonal fashion suggestions
* 🛍 Shopping redirects to Amazon, Flipkart, and Myntra
* ⚡ Interactive UI with drag-and-drop image upload
* 🎯 Smart style recommendations using AI

## ⚙ Installation

### 1. Clone the repository

```
git clone https://github.com/badigevamshi/AI-Fashion-Recommendation-System.git
cd AI-Fashion-Recommendation-System
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Create `.env` file

```
GROQ_API_KEY=your_api_key_here
```

### 4. Run the application

```
python app.py
```

### 5. Open in browser

```
http://127.0.0.1:5000
```
